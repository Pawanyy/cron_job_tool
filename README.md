
# Website Cron Job Runner

A tool written in node js using node-cron, axios, logger to logout results of web request.

Mainly use for website cron request in local the project can be modified as per requirement.

## Run Locally

Clone the project

```bash
  git clone https://github.com/Pawanyy/cron_job_tool
```

Go to the project directory

```bash
  cd cron_job_tool
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  node index.js
```
